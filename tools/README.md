# tools/

Utilities deployed to `C:\Tools\` during Phase 3.

| File | Purpose |
|------|---------|
| `winrar-x64-701.exe` | WinRAR silent installer |
| `nssm-2.24.zip` | Non-Sucking Service Manager (service helper) |
| `sysinternals/procexp64.exe` | Process Explorer — debugging |
| `sysinternals/Procmon64.exe` | Process Monitor — debugging |
| `sysinternals/handle64.exe` | Handle viewer — debugging file locks |
| `sysinternals/PSTools.zip` | PsExec, PsList, etc. — remote admin tools |

SysInternals and tshark are included because post-deployment downloads are impossible in the air-gapped environment.
